How to Use Checkify

Quick Start

1 Login to Logistics Portal - Make sure you're logged into the logistics portal in your browser.
2 Login to Vehicle API - Enter your username and password in the header and click Login.
3 Select Date - Choose the date you want to fetch routes for.
4 Fetch Journeys - Click the Fetch Journeys button to retrieve routes from Amazon Logistics.
5 Create Reviews - Use the action buttons to create vehicle reviews.
