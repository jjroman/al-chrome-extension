üìù Submitting Reviews

When you click the Review button, the extension will:

  • Check if vehicle exists - If not, you'll be prompted to create it with Make, Model, Year, and Type.
  • Check if driver exists - If not, you'll be prompted to create them with their contact info.
  • Handle multiple drivers - If multiple drivers share the same phone/email, the system matches by name.
  • Prompt for review details - Enter notes and select damage positions.

üöó Creating Vehicles

When a vehicle doesn't exist, you'll need to provide:

  • Make - e.g., Ford, Mercedes, RAM
  • Model - e.g., Transit, Sprinter, ProMaster
  • Year - e.g., 2024
  • Type - e.g., Van, Truck, Car

üë§ Creating Drivers

Driver information is pre-filled from Amazon Logistics data. You can confirm or edit:

  • First Name & Last Name
  • Phone Number (required)
  • Email (optional, for notifications)
üí° Tip: The driver will receive an email notification with a link to complete the vehicle review if their email is available.

üìç Damage Positions
